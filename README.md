# Лабораторная работа №5

## Задание 1. Склад

Реализована программа для учёта товаров на складе. Поддерживаются команды:

- `ADD <товар> <количество> <адрес>` — добавить товар в ячейку
- `REMOVE <товар> <количество> <адрес>` — удалить товар из ячейки
- `INFO` — вывести состояние склада

**Параметры склада:** 3 зоны, 14 стеллажей, 4 секции, 8 полок.

## Задание 2. Электронная очередь

Реализовано моделирование работы электронной очереди. Поддерживаются команды:

- `ENQUEUE <время>` — добавить посетителя в очередь
- `DISTRIBUTE` — распределить посетителей по окнам

Распределение выполняется жадным алгоритмом: каждый посетитель направляется в окно с минимальной загрузкой.

## Задание 3. Маршруты самолётов

Реализована система хранения маршрутов самолётов. Поддерживаются команды:

- `CREATE_PLANE <имя> <город1> ... <городN>` — создать самолёт
- `PLANES_FOR_TOWN <город>` — показать самолёты над городом
- `TOWNS_FOR_PLANE <имя>` — показать маршрут и другие самолёты
- `PLANES` — показать все самолёты

## Задание 4. Расписание занятий

Реализована система учёта расписания. Поддерживаются команды:

- `CLASS <день> <дисциплина>` — добавить занятие
- `NEXT` — перейти на следующий месяц
- `VIEW <день>` — посмотреть расписание на день

При переходе на следующий месяц дни, которых нет в новом месяце, переносятся на последний день.
